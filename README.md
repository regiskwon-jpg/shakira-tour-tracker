## ✨ Features

| Tab | What it does |
|---|---|
| **📧 Emails** | Browse all tracked email threads. Filter by category. Expand to see action items. |
| **✅ Actions** | Flat list of all action items across every email, grouped by status. |
| **＋ Log Email** | Paste a new email — AI auto-extracts summary, category, tags, and action items. |
| **✦ Ask AI** | Natural language queries against the full tracker (e.g. "What's still unsigned?"). |

### Status Tracking
Each action item can be set to:
- 🔴 **Open** — not started
- 🟡 **In Progress** — being worked on
- 🟠 **Blocked** — waiting on someone or something
- 🟢 **Done** — completed

## 🔑 AI Features (Optional)

The **Parse Email** and **Ask AI** features use the Anthropic Claude API.

To enable:
1. Click the **🔑 API Key** button in the top right of the app
2. Enter your Anthropic API key (`sk-ant-...`)
3. The key lives only in your browser session — it is never stored or transmitted elsewhere

Get an API key at: [console.anthropic.com](https://console.anthropic.com)

> All other features (browsing, filtering, status updates) work fully offline with no API key.
